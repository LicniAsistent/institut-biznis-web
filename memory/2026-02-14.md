# 2026-02-15 - Course Channels & Navigation

## NEW: Kursevi Button & Course Channels

### Kursevi Button (Purple)
**Location:** Community page, left sidebar, top
- **Color:** Gradient purple-to-pink (`from-purple-600 to-pink-600`)
- **Icon:** BookOpen
- **Action:** Vodi na `/courses` stranicu

### Course Channels System
**Location:** Community page, left sidebar, collapsible section

**How it works:**
1. User purchases a course → `purchasedCourses: ['auto-perionica']`
2. Course channels appear in "Moji Kursevi" section
3. Channels are PURPLE themed (vs general channels which are primary/green)
4. Each course has multiple channels:
   - Uvod u kurs
   - Oprema i alati
   - Tehnika pranja
   - Detailing
   - Praksa i saveti
   - Moji projekti

**Visual indicators:**
- Channel icon: Hash (purple when active)
- Banner above chat: Purple background with "KURS" badge
- Channel name has purple accent

### User Demo Data
```javascript
purchasedCourses: ['auto-perionica']  // User bought this course
// 'dostava' course exists but not purchased
```
